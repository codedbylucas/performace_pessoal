# Painel de Performance Pessoal (100% offline/sem IA)

Aplicação full-stack focada em performance pessoal, sem necessidade de chaves de IA.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Configure `.env` com `SUPABASE_URL` e `SUPABASE_SERVICE_ROLE_KEY` (já exemplificados em `.env.example`).
3. Rode o SQL de `supabase.sql` no SQL Editor do Supabase para criar tabelas.
4. Run the app:
   `npm run dev`
